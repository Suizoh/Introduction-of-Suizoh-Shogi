# 醉象将棋（Suizo Shogi）

> 全てを守れないとき、あなたは何を選ぶか。
>
> 一人ぼっちだった王に、息子が加勢した将棋。

---

## なぜ「ひとりぼっち」なのか

※ 本作では太子（倅・せがれ）と表すことがある。醉象が成ると、その醉象は太子となる。

将棋には、取った駒を自分の駒として再利用できるという特徴がある。
合理的で、優れた仕組みだ。だがそれは同時に、こうも言える。

昨日の友は今日の敵——対局開始時に味方だった駒が、次の瞬間には敵として自分に刃を向けてくる。それが将棋という世界では、極々普通に、日常茶飯事として起こる。
飛車も、角も、金も銀も、例外ではない。

王はそれを知っている。安易に誰も頼る・信頼することができない。
一瞬の油断が命取り。

味方に囲まれ、最高権力を手にしているはずの王は——その実、ひとりぼっちだ。

だが、醉象や太子（倅）だけは違う。
取っても再利用できない。つまり、どう転んでも王に刃を向けることがない。
敵からすれば「煮ても焼いても食えない」。盤上で唯一、絶対に裏切らない存在——それが醉象であり、倅である。

**この将棋は、そこに「絶対に裏切らない存在」を加えたとき、何が変わるのかを問いかける。**

---

## これはどんなゲームか

**醉象将棋**は、通常の将棋に「醉象（すいぞう）」という駒を加えた変則将棋シリーズです。

醉象は真後ろにだけ動けない駒で、敵陣に入ると**倅（せがれ）＝太子**に成ります。
太子は玉将・王将と完全に等価な駒であり、一度太子が誕生した陣営は
**玉と太子の両方を失うまで敗北しません**。

醉象・太子は**再使用不可**です（玉と同じ扱い。相手から取っても持ち駒にはなりません）。

この一枚の駒が加わることで、将棋は「王を守る戦い」から
「王と息子、二人を守る戦い」へと変わります。

## バリアント一覧

醉象将棋シリーズには、醉象の初期配置・導入方法とオプションルールの組み合わせにより、
現在 **10種類のバリアント**が存在します。

### バージョン（醉象の初期配置・導入方法）

| バージョン | 概要 |
|---|---|
| **０** | 玉の代わりに醉象を使う。醉象そのものが王族駒（王手の対象）となる変則将棋 |
| **１** | 先手の9九・後手の1一に醉象を配置して開局（左香を一歩前進させてスペースを確保） |
| **２** | 醉象を持ち駒としてスタート。指定手数を過ぎるまで使用不可。打込は自陣の空マスのみ |
| **３** | 先手は5・6段目、後手は4・5段目の任意マスに醉象を配置して開局（後手が先決・後決を選択） |

### タイプ（オプションルール）

| タイプ | テーマ | 概要 |
|---|---|---|
| **O** | 王と太子による共同運営 | オプションなし。醉象は王族駒にならない。玉と倅のみが王族駒 |
| **A** | 太子の擁立 | 醉象も王族駒。先に倅を誕生させた方が勝利。玉か醉象が詰んだ時点でも終局 |
| **B** | 王位継承・世代交代 | 「約束の手数」を過ぎると両者の玉が没収され、倅（または醉象）による継戦に移行 |

醉象将棋０はタイプの区別なし（専用ルール）。
１〜３はA・B・Oの3タイプと組み合わせ可能で、合計 **1 + 3×3 = 10バリアント**。

## 醉象という駒について

醉象は古代将棋（中将棋・大将棋など）にも登場する駒です。
本作での駒の動きはオリジナルに準拠していますが、その他の扱いについては
オリジナルと異なる点がある可能性があります。

なお、5八・5二に配置する類似の変則将棋として「朝倉将棋」が歴史上実在しています。
本作はそれとは異なるものです。
